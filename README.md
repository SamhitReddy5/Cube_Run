# Cube Run (Unity Game)

Cube Run is a 3D endless runner-style game developed using Unity. The player controls a moving cube and must avoid obstacles while progressing through the level. The project demonstrates core game development concepts such as player movement, collision handling, and level design.

## Features

* Continuous forward movement (endless runner mechanic)
* Obstacle avoidance and collision detection
* Simple and responsive player controls
* Basic level design with increasing challenge
* Game over logic on collision

---

* Unity (Game Engine)

## How to Run

### Play the Game

1. Open the `Build/` folder
2. Run the `.exe` file

### Open in Unity

1. Open Unity Hub
2. Click "Add Project"
3. Select this folder
4. Open the project and run the main scene
